Train 2022

This is an example of an Angular Web-App that I created as a practice exercise for a course. This is a simulation of searching and buying a train ticket. It was made using Bootstrap, HTML, CSS and TypeScript, it contains filtering and sorting options. It is possible to buy a new ticket and also delete a purchased ticket.

This is a web application made in the Angular framework, made for a Cakes Shop. The application can filter by ingredients of Cakes.Cakes are dynamically added to the list of Cakes. The user can create their own profile and send a message to the server.

Art Museum

This is a web application made in Angular using Bootstrap, HTML, CSS, and TypeScript. Here exhibitions of a Museum are displayed, new art can be added and existing ones can be deleted from the exhibition. By filling out the form, a new exhibition is added to the Museum.
